This is the OpenGLib library by Morteza Mostajab
It is used to create the window for the opengl easier and handle opengl operations easier.

To use this project, you need to have Cmake, OpenGl and GLFW

Build Project:
1. mkdir build
2. cd build
3. cmake .. or ccmake ..

Supported features:

* Immediate Mode Rendering
* Vertex Arrays
* Vertex Buffers
* Textures
* Shader Programming
